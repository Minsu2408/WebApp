## 2. 교과목 개요 및 목표
본 교과목은 기초 웹 프로그래밍(HTML, CSS, 자바스크립트) 지식을 바탕으로, 현대적인 웹 애플리케이션 개발에 필요한 심화 기술을 습득하는 것을 목표로 합니다. 
Front-end와 Back-end 프레임워크의 구조를 이해하고, 데이터베이스 연동 및 RESTful API 설계를 통해 실무에서 활용 가능한 풀스택(Full-Stack) 웹 애플리케이션 개발 역량을 배양합니다.

### 주요 학습 목표
1. **현대적 프론트엔드 프레임워크** (예: React, Vue 등)의 컴포넌트 기반 개발 방식 이해 및 구현 능력을 갖춘다.
2. **백엔드 프레임워크** (예: Node.js/Express, Spring Boot 등)를 활용한 서버 아키텍처 및 REST API를 설계할 수 있다.
3. **데이터베이스(SQL / NoSQL)**와의 연동을 통해 동적 데이터를 처리하고, 사용자 인증(JWT, OAuth) 시스템을 구축할 수 있다.
4. **팀 프로젝트**를 통해 요구사항 분석, 설계, 구현, 배포(Cloud/DevOps 기초)까지의 전체적인 웹 서비스 생명 주기를 경험한다.
