# 리액트 프로그래밍 학습 정리 (03)

## 1. JSX (JavaScript XML)
- **정의**: 자바스크립트를 확장한 문법으로, XML과 유사한 구문을 사용하여 DOM 트리를 생성함.
- **특징**: 브라우저 실행 전 **바벨(Babel)**을 통해 일반 자바스크립트로 변환됨.
- **장점**: 가독성이 높고 자바스크립트와 HTML을 동시에 작성할 수 있어 편리함.



## 2. 가상 DOM (Virtual DOM)
- 리액트 엘리먼트를 실제 브라우저 DOM에 직접 렌더링하기 전 거치는 단계.
- **동작 방식**: 데이터 변경 시 가상 DOM 전체를 업데이트한 후, 실제 DOM과는 **차이가 있는 부분만** 반영하여 성능을 최적화함.

## 3. 실습 예제: 바이너리 시계
- 매초 `update()` 함수를 호출하여 현재 시각을 2진수로 변환.
- `ReactDOM.render()`를 통해 변경된 부분의 DOM만 효율적으로 업데이트함.

## 4. 핵심 요약
- JSX 내 중괄호 `{...}`를 사용하면 자바스크립트 표현식을 삽입할 수 있음.
