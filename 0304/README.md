# 웹 프로그래밍 기초 학습 정리 (01)

## 1. HTML (Hyper Text Markup Language)
- **정의**: 웹사이트의 구조를 기술하기 위한 마크업 언어 [[2](https://gist.github.com/allysonsilva/85fff14a22bbdf55485be947566cc09e)].
- **구조**: `<html>`, `<head>`, `<body>` 태그로 구성된 문서 구조를 가짐.

## 2. CSS (Cascading Style Sheet)
- **역할**: HTML 문서의 디자인과 스타일을 담당하는 시트 언어.

## 3. 웹 서버 환경 설정 및 테스트
- **Tomcat**: Java 기반 웹 서버(JSP 등) 테스트 방법.
- **Apache (Laragon)**: PHP 파일(`hello.php`)을 이용한 서버 구동 실습.

## 4. 리액트(React) 개요
- **특징**: UI를 만들기 위한 자바스크립트 라이브러리.
- **주요 개념**: JSX, 컴포넌트, 생명주기, 가상 DOM.

- Express 모듈 (Node.js) 등
