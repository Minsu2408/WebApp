# 리액트 프로그래밍 학습 정리 (04)

## 1. 컴포넌트(Component)의 이해
* **정의**: 입력(props)을 받아 엘리먼트(Element)를 출력하는 재사용 가능한 독립 모듈.
* **특징**: 레고 블록처럼 조립하여 UI를 구축하며, 자바스크립트 함수와 유사하게 작동함.

## 2. Props와 데이터 전달
* **Props**: 컴포넌트에 전달되는 입력값.
* **실습 내용**: `Comment` 컴포넌트에 이름(`name`)과 내용(`comment`)을 props로 전달하여 화면에 표시.

## 3. 컴포넌트의 구성 및 반복
* `comments` 배열 객체를 생성하여 데이터 분리.
* `map()` 함수를 사용하여 다수의 `Comment` 컴포넌트를 동적으로 생성.

## 4. 과제 및 발표 주제
* Pure function vs Impure function
* 화살표 함수(Arrow function) 및 람다 함수
* `map()` 함수의 활용
