# 리액트 프로그래밍 학습 정리 (02)

## 1. 리액트(React)란?
- 사용자 인터페이스(UI) 구축을 위한 자바스크립트 라이브러리 [[2](https://docs.github.com/github/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)].
- 가상 DOM(Virtual DOM)과 JSX 방식을 사용하며, SPA 개발에 최적화됨.
- **컴포넌트** 단위로 고립된 코드 파편을 조합하여 복잡한 UI 구성.

## 2. 주요 장점
- 컴포넌트 재사용성 및 유지보수 용이성.
- 거대한 커뮤니케이션과 다양한 툴 지원.
- React Native를 통한 모바일 앱 확장성.

## 3. 프로젝트 구조 및 렌더링
- `App.js`: 실제 사용자에게 보여줄 화면 정의.
- `index.js`: `root` 요소에 `App.js`를 렌더링.
- `index.html`: `root`라는 ID를 가진 `div` 포함.

## 4. 과제 및 발표 주제
- **Report**: 깃허브 디렉토리에 build 호스팅.
- **발표**: Git & SourceTree 사용법 (commit, branch, merge 등).
