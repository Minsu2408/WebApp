React의 useState Hook을 활용하여 상태 관리를 구현하는 기초 방법을 학습했습니다.
Counter (number): 숫자 상태를 정의하고, 버튼 클릭 시 setCount(count + 1)을 통해 값을 1씩 증가시키는 가장 기본적인 상태 업데이트 패턴입니다.

Text field (string): 입력창의 문자열을 상태로 관리하여 실시간으로 텍스트를 변경하는 예제입니다.

Checkbox (boolean): 체크 여부(true/false)를 논리값 상태로 제어합니다.

Form (two variables): 두 개 이상의 독립적인 상태 변수를 하나의 컴포넌트에서 동시에 관리하는 복합적인 구조를 다룹니다.

이러한 개별 컴포넌트들을 작성한 뒤, 공통 컴포넌트 관리 원칙에 따라 App.js에 임포트하여 통합하는 과정을 거쳤습니다. 상태(State)가 변할 때마다 React가 UI를 어떻게 리렌더링하는지 이해하는 것이 이번 수업의 핵심입니다.
